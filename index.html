My NSS Weekly Blog

Week July 9-15

This was my first full week of class at NSS. Last week we spent time going over basic html and a little bit of css. Most of that information was not new to me but was a decent refresher. We also set up some package managers in terminal, plus zsh. I'm still not quite sure what exactly zsh does, but I'm sure I'll have a better understanding later.

This week we went over basic javascript. Variables, Arrays, Conditionals, Loops, and Functions. We did one exercise building a happy number calculator, which took me awhile to fully grasp what everything was doing. Still a little foggy. We also did our first group project this week, which was a good exercise in getting used to working with a team on Github. I feel that I've got a good understanding of Github so far. Next week we're going to use the projects feature of Github. Tickets, etc.

July 17-21

This second week was about understanding javascript even further. We went over event listeners and more referencing to html elements. We did our second group project on Monday. It had the same requirements as the first one, except we had only one day to do it. The programming went a lot faster and my css ended up being a lot cleaner.

On Tuesday we started our javascript challenge, which was to build a triangle in the console log with a user-defined height and character. It definitely is challenging and requires nesting loops inside of a main loop. A fellow student showed me how to use the debugger to pause and click through the programming one loop at a time. It definitely helps.

Thursday we were introduced to sass and bootstrap. Sass definitely will make css editing a lot easier and cleaner. I grasped most of the concepts of sass right away. Bootstrap is taking me a longer time to use since I've got to look up each class on their site for every element I use. This blog will eventually be built on bootstrap
