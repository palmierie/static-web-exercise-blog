My NSS Weekly Blog

Week July 9-15

This was my first full week of class at NSS. Last week we spent time going over basic html and a little bit of css. Most of that information was not new to me but was a decent refresher. We also set up some package managers in terminal, plus zsh. I'm still not quite sure what exactly zsh does, but I'm sure I'll have a better understanding later.

This week we went over basic javascript. Variables, Arrays, Conditionals, Loops, and Functions. We did one exercise building a happy number calculator, which took me awhile to fully grasp what everything was doing. Still a little foggy. We also did our first group project this week, which was a good exercise in getting used to working with a team on Github. I feel that I've got a good understanding of Github so far. Next week we're going to use the projects feature of Github. Tickets, etc.
